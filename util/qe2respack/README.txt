# Usage of qe2respack (an interface from Quantum Espresso to RESPACK)

Usage example: "python qe2respack.py ./outdir/prefix.save"
   ./outdir/prefix.save is the Quantum Espresso output directory.
   qe2respack.py will create dir-wfn directory under a directory where qe2respack.py is called.
